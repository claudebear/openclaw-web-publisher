# 🦞 OpenClaw Setup Guide {#openclaw-setup-guide .text-3xl .font-bold .text-slate-900}

Target OS: Ubuntu 24.04 / 22.04 LTS (Updated February 2026)
:::

::: {.section .mb-10}
## 1. Prerequisites {#prerequisites .text-xl .font-semibold .text-slate-800 .mb-4}

OpenClaw requires Node.js 22 or higher. Ensure your system is up to
date.

    # Update system packages
    sudo apt update && sudo apt upgrade -y

    # Install Node.js (via NodeSource)
    curl -fsSL https://deb.nodesource.com/setup_22.x | sudo -E bash -
    sudo apt install -y nodejs
:::

::: {.section .mb-10}
## 2. Core Installation {#core-installation .text-xl .font-semibold .text-slate-800 .mb-4}

Install the OpenClaw CLI and the Gateway service globally.

    sudo npm install -g openclaw@latest
:::

:::: {.section .mb-10}
## 3. Configuration (Google Gemini) {#configuration-google-gemini .text-xl .font-semibold .text-slate-800 .mb-4}

Run the onboarding wizard to link your API key and WhatsApp.

    openclaw onboard

::: {.bg-blue-50 .border-l-4 .border-blue-400 .p-4 .my-4}
**Note:** Select **Gemini 2.5 Flash** or **3.0 Flash** for the best
performance. When prompted for WhatsApp, a QR code will appear in your
terminal---scan it with your phone.
:::
::::

::: {.section .mb-10}
## 4. Running as a Service (Ubuntu) {#running-as-a-service-ubuntu .text-xl .font-semibold .text-slate-800 .mb-4}

To ensure the bot starts automatically when Ubuntu boots, install it as
a systemd daemon.

    # Install and start the background service
    openclaw service install
    openclaw service start

    # Useful Management Commands
    openclaw service status
    openclaw service restart
    openclaw logs --follow
:::

::: {.section .mb-10}
## 5. Enabling Research & Internet Access {#enabling-research-internet-access .text-xl .font-semibold .text-slate-800 .mb-4}

Open the configuration file to enable **Google Search Grounding**.

    nano ~/.openclaw/openclaw.json

Update the `tools` section as follows:

    "tools": {
      "google_search": { "enabled": true },
      "web_fetch": { "enabled": true }
    }
:::
